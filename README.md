# StudCenter App

Mobile app for the university student center with electronic queue management. Built with Kotlin Multiplatform and Compose Multiplatform.

## Features

- University email authentication with OTP verification
- Electronic queue management for student center visits
- Real-time queue status tracking
- Appointment booking system

## Tech Stack

| Layer | Technology |
|-------|-----------|
| UI | Compose Multiplatform |
| Architecture | MVI + Clean Architecture |
| Networking | Ktor |
| Dependency Injection | Koin |
| Async | Coroutines + Flow |

## Status

Core authentication flow and main queue screens implemented. Project developed as part of a university initiative.
